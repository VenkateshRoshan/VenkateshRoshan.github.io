MedhAItron - Neuro Assistive Robotics Initiative

EEG + BCI + AI/ML + Exoskeletons
Learn More About MedhAItron

MedhAItron is my startup-focused project to build an intelligent neuro-assistive system that translates brain activity (EEG) into real-time control of a robotic exoskeleton. The goal is to help people with severe motor impairments regain functional movement through a safe, adaptive, and responsive brain-computer interface.

The platform combines high-resolution EEG acquisition, signal preprocessing, feature extraction, and deep-learning-based neural decoding to convert intent into motion commands. These commands are then executed through an AI-assisted planning and control pipeline to deliver smooth and stable exoskeleton movement aligned with the user's intended action.

My long-term vision is to build a personalized closed-loop rehabilitation system that improves over time using biosignal feedback, sensor fusion, and adaptive learning.

DriveAlign - Autonomous Driving with RLHF

PPO + RLHF + MetaDrive + PyTorch + CNN + MLP

Built a full Reinforcement Learning from Human Feedback (RLHF) pipeline for autonomous driving inside the MetaDrive simulation environment. Trained a multimodal PPO agent from scratch using a custom gated CNN + MLP architecture. The CNN branch processes live RGB camera frames while the MLP branch processes a stacked history of 4 ego-state vectors covering speed, steering angle, yaw rate, lateral offset, and boundary distances. A learned gate layer dynamically weights how much to trust vision versus scalar signals depending on the driving context.

Implemented a complete 7-phase RLHF pipeline: base PPO agent, vision integration, segment recording, human feedback collection, reward model training, RLHF fine-tuning, and metrics visualization. Human preferences are collected by watching the agent drive in real time and scoring each segment.

Rufus - Agentic Web Data Extractor for RAG Systems

🤖 Engineered a cutting-edge AI-powered web extraction system that revolutionizes how data is gathered and processed for RAG (Retrieval-Augmented Generation) applications. Rufus intelligently crawls websites based on natural language instructions, making complex data extraction as simple as writing a prompt.

Key Features

  • AI-Driven Crawling: Advanced crawling algorithms powered by LLMs (TinyLlama, with planned OpenAI integration).
  • Smart Content Selection: Intelligent filtering mechanisms that identify and extract only the most relevant content.
  • Dynamic Navigation: Robust handling for complex web structures, including nested links and dynamically loaded content.
  • Structured Output: Sophisticated document synthesis pipelines that convert raw web content into clean, structured formats.

Technical Implementation

  • Core Architecture: Built with Python, FastAPI for REST endpoints and Gradio for UI. Containerized with Docker.
  • LLM Integration: Leveraged LangChain for orchestrating LLM operations.
  • Vector Storage: ChromaDB for efficient storage and retrieval of document embeddings.
